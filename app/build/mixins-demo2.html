Mixin block with Rest Arguments

    Type of Mixins

  • Without Arguments

    mixin list

    +list

  • With Arguments

    mixin pet(name)

    +pet('cat')

  • Mixin Blocks

    mixin article(title)

    +article('Hello world')

  • With unknown number of arguments

    mixin list(id, ...items)

    +list('my-list', 1, 2, 3, 4)
